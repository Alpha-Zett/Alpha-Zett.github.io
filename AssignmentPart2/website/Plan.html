Technologies

Collaborative Workspaces

Microsoft Teams

We use Microsoft Teams to communicate with each other and posting updates. All our meetings and recordings are also on Microsoft Teams.

Github

Our website and files are stored on Github so everyone gets access to them and can make changes.

Trello

We use Trello to list things to do and keep track of progress for our tasks.

Software

Visual Studio Code

Website coding is mainly performed on Visual Studio Code which is one of the most convenient softwares to code as it can define your coding language and make auto adjustments as you code.

Lucid Chart

Lucid Chart is a website-based application which allows users to create and share charts and diagrams. This will be used to create visual plans for our features.

Figma

Figma is a website-based application used to create prototypes. This will be used to create our feature designs and prototypes.
