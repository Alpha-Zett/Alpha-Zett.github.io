viCOVID-19 website

This is a website containing information about COVID-19 within Victoria and has been set up to provied up to date information about where COVID-19 is spreading so that people can be more informed of where hot spots are, where testing facilities are, quick access to news feeds and easy access to a chatbot. There is a Contact Tracing registration page and a trend forecasting and a break down of cases on the Case Presentation Page.
